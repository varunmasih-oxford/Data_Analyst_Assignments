# MS Excel -- Page Layout Tab

## Introduction

The Page Layout tab in Microsoft Excel is used to control how a
worksheet looks when printed.\
It helps manage themes, margins, orientation, size, print area, scaling,
sheet options, and arrangement.

------------------------------------------------------------------------

# 1. Themes Group

Themes control the overall design of the workbook.

### 1.1 Themes

-   Apply a complete design (colors, fonts, effects).

### 1.2 Colors

-   Change theme color combinations.

### 1.3 Fonts

-   Change heading and body fonts.

### 1.4 Effects

-   Apply visual effects to shapes and charts.

------------------------------------------------------------------------

# 2. Page Setup Group

This group controls print settings.

### 2.1 Margins

-   Normal
-   Wide
-   Narrow
-   Custom Margins

### 2.2 Orientation

-   Portrait (Vertical)
-   Landscape (Horizontal)

### 2.3 Size

-   A4
-   Letter
-   Legal
-   Custom Size

### 2.4 Print Area

-   Set Print Area
-   Clear Print Area

### 2.5 Breaks

-   Insert Page Break
-   Remove Page Break
-   Reset All Page Breaks

### 2.6 Background

-   Add background image (not printed).

### 2.7 Print Titles

-   Repeat Rows at Top
-   Repeat Columns at Left

------------------------------------------------------------------------

# 3. Scale to Fit Group

Used to adjust worksheet size to fit printed pages.

### 3.1 Width

-   Fit to 1 page wide
-   Automatic

### 3.2 Height

-   Fit to 1 page tall
-   Automatic

### 3.3 Scale

-   Adjust scaling percentage (e.g., 100%, 80%)

------------------------------------------------------------------------

# 4. Sheet Options Group

Controls gridlines and headings visibility.

### Gridlines

-   View (show/hide)
-   Print (print gridlines)

### Headings

-   View row/column headings
-   Print row/column headings

------------------------------------------------------------------------

# 5. Arrange Group

Used for arranging objects like images and shapes.

### 5.1 Bring Forward

-   Bring to Front
-   Bring Forward

### 5.2 Send Backward

-   Send to Back
-   Send Backward

### 5.3 Selection Pane

-   Manage objects visibility

### 5.4 Align

-   Align Left/Right/Top/Bottom
-   Center alignment

### 5.5 Group

-   Group objects
-   Ungroup objects

### 5.6 Rotate

-   Rotate Right
-   Rotate Left
-   Flip Vertical/Horizontal

------------------------------------------------------------------------
