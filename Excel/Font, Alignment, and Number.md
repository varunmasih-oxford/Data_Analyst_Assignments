# Assignment: Font, Alignment, and Number Formatting  

---

## 1. Data Entry Task

Create the following table in **MS Excel**:

| Employee ID | Employee Name | Department | Basic Salary | HRA | Bonus | Total Salary |
|------------|---------------|------------|--------------|-----|-------|--------------|
| E101 | Rahul Sharma | Sales | 25000 | 5000 | 3000 | |
| E102 | Neha Verma | HR | 30000 | 6000 | 3500 | |
| E103 | Amit Patel | IT | 40000 | 8000 | 5000 | |
| E104 | Priya Singh | Accounts | 28000 | 5600 | 2500 | |

### Calculation Rule
```
Total Salary = Basic Salary + HRA + Bonus
```

---

## 2. Font Formatting Requirements

### 2.1 Report Title
- Text: **Monthly Salary Report – ABC Solutions Pvt. Ltd.**
- Font Size: 16 or 18
- Font Style: Bold
- Font Color: Dark Blue
- Merge and center the title across the table

### 2.2 Table Header Row
- Font Style: Bold
- Font Color: White
- Background Fill: Dark Blue or Dark Gray
- Font Size: 11 or 12

### 2.3 Employee Name Column
- Font Style: Italic

---

## 3. Alignment Formatting Requirements

| Data Type | Alignment |
|---------|-----------|
| Report Title | Center (Horizontal & Vertical) |
| Employee ID | Center |
| Department | Center |
| Employee Name | Left |
| Salary Values | Right |
| All Rows | Vertical Center |

Additional Instructions:
- Use **Wrap Text** for column headings if required
- Ensure all data is clearly visible without overlapping

---

## 4. Number Formatting Requirements

### 4.1 Salary Columns
(Basic Salary, HRA, Bonus, Total Salary)

- Format Type: Currency
- Currency Symbol: ₹ (Indian Rupee)
- Number Style: Comma-separated
- Decimal Places: 0

**Example:**
```
₹25,000
```

### 4.2 Employee ID Column
- Format as Text
- Ensure IDs are not auto-converted

---

## 5. Final Formatting Standards

- Apply **All Borders** to the table
- Use **AutoFit Column Width**
- Highlight **Total Salary** column using:
  - Bold font
  - Light background color
- Apply a **Bottom Border** to the header row
