# Sparklines in Excel

## What is a Sparkline?

A **Sparkline** is a small chart that appears inside a single cell in Excel.  
It is used to quickly show **trends or patterns in a dataset** without creating a full chart.

Sparklines are useful for:
- Sales reports
- Financial summaries
- Performance tracking
- Quick trend visualization

---

## Types of Sparklines

Excel provides three main types:

1. **Line Sparkline**
   - Shows trends using a small line graph.
   - Best for time-series data like monthly sales.

2. **Column Sparkline**
   - Displays data as small vertical bars.
   - Useful for comparing values.

3. **Win/Loss Sparkline**
   - Shows only positive or negative values.
   - Often used for game results or profit/loss tracking.

---

## Steps to Insert a Sparkline

1. Select the cell where you want the sparkline.
2. Go to the **Insert** tab.
3. Choose **Sparkline** and select a type (Line, Column, Win/Loss).
4. Select the data range.
5. Click **OK**.
